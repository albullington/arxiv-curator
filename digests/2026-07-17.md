## [Bridge Evidence: Static Retrieval Utility Does Not Predict Causal Utility in Multi-Step Agentic Search](https://arxiv.org/abs/2607.15253v1)
**Score:** 0.959  |  **arXiv:** 2607.15253v1

This paper reveals that traditional, static measures of document relevance are virtually uncorrelated with a document's actual causal usefulness in multi-step, agentic search. Through counterfactual testing on a language model agent, the authors identify "bridge documents"—texts that appear useless to a static evaluator but are actually crucial because they contain key entities that guide the agent’s subsequent queries. These findings demonstrate that optimizing retrieval systems for static relevance fails to improve multi-step agentic performance, highlighting a critical gap in how we evaluate search tools for AI agents.

## [TRACE: Turn-level Reward Assignment via Credit Estimation for Long-Horizon Agents](https://arxiv.org/abs/2607.13988v1)
**Score:** 0.927  |  **arXiv:** 2607.13988v1
**Length:** 26 pages

To address the credit-assignment challenge in long-horizon agent tasks, this paper introduces TRACE, a dense reinforcement learning method that assigns turn-level rewards to individual tool-use actions. Instead of relying on sparse outcome rewards or training a separate critic model, TRACE derives per-step rewards from temporal-difference changes in gold-answer log-probabilities using a frozen reference model. When evaluated on complex search benchmarks, this approach dramatically boosts agent performance and training convergence without requiring cold-start supervised fine-tuning.
