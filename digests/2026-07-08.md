**Score:** 0.636  |  **arXiv:** 2607.06527v1

RSF-GLLM tackles the challenge of multi-hop Knowledge Graph QA where intermediate nodes lack lexical overlap with the query by decoupling differentiable graph reasoning from answer generation. Its Recurrent Soft-Flow (RSF) module propagates continuous relevance scores, leveraging structural cues to traverse semantically dissimilar "bridge" nodes and derive discrete reasoning paths. These paths then fine-tune a Large Language Model for grounded answer generation, achieving competitive performance with superior inference efficiency compared to other LLM-based approaches.

## [FreqDepthKV: Frequency-Guided Depth Sharing for Robust KV Cache Compression in Long-Context LLM Inference](https://arxiv.org/abs/2607.06519v1)
**Score:** 0.631  |  **arXiv:** 2607.06519v1

FreqDepthKV addresses the significant memory and bandwidth overhead of KV caches in long-context LLMs. It compresses KV states by factorizing adjacent layers into shared, low-frequency depth components and sparse, high-frequency residuals, dynamically assigning attention heads to optimal compression modes based on their importance. This method significantly reduces memory and improves inference throughput while preserving accuracy across various long-context tasks, notably outperforming prior compression techniques.
